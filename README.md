🎙️ Multilingual Speech-to-Image Generation

Convert speech from any language into AI-generated images using Whisper, Translation APIs, and Stable Diffusion.

## 🚀 Features

- 🎤 Speech-to-Text using OpenAI Whisper
- 🌍 Supports multiple languages
- 🔄 Automatic translation to English
- ✨ Prompt enhancement for better image quality
- 🎨 AI Image Generation using Stable Diffusion
- 💾 Saves generated images automatically
- 🖥️ Compatible with Google Collab

Task-3:
🎙️ Voice Activated Personal Assistant

A Python-based Voice Activated Personal Assistant that can listen to voice commands, respond using speech synthesis, provide weather updates, read news headlines, tell the current time, and set reminders.

## 🚀 Features

- 🎤 Voice Recognition using SpeechRecognition
- 🔊 Text-to-Speech Responses using gTTS
- 🕒 Current Time Information
- 🌦️ Real-Time Weather Updates
- 📰 Latest News Headlines
- ⏰ Reminder System
- ⌨️ Supports Both Voice and Text Commands
- 🖥️ Compatible with Python 3.13.5
- 💻 Runs on Google Collab
